# User Manual

## Using the Web Interface
1. Access `http://127.0.0.1:5000/`
2. Fill all fields:
   - Age: Patient age in years
   - Gender: Male/Female
   - Medical values: Use normal ranges as guide

## Interpreting Results
- High Risk (Red): >65% probability
- Medium Risk (Yellow): 35-65% probability 
- Low Risk (Green): <35% probability
